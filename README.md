# México Medieval

Website for [México Medieval](https://www.mexicomedieval.org) — articles, books, papers, events, podcasts, and related content.

## Documentation

All guides live on the **`dev`** branch:

**[→ Documentation hub (`landing/docs/README.md`)](landing/docs/README.md)**

That hub links to content-editing guides, local development setup, frontend conventions, and how deployment from `dev` to GitHub Pages works.

## Branches

| Branch | Purpose |
|--------|---------|
| **`dev`** | Source of truth — edit here |
| **`main`** | Auto-generated static site for GitHub Pages — do not edit manually |

See [Deployment and branches](landing/docs/deployment.md) for details.

## Quick start (developers)

```bash
cd landing
pnpm install
pnpm run dev
```

More in [Local development](landing/docs/local-development.md).
